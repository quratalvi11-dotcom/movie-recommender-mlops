# Movie Recommender MLOps

An end-to-end MLOps project that recommends movies to users based on their rating history, covering the full machine learning lifecycle from data ingestion to automated retraining.

Pick a user, see what they have rated, and get a top-10 recommendation list generated by a collaborative filtering model served through a reproducible, monitored, CI/CD-driven pipeline.

## MLOps lifecycle coverage

-  Data validation (Great Expectations)
-  Data versioning (DVC)
-  Feature engineering and feature store (Feast)
-  Experiment tracking (MLflow)
-  Hyperparameter tuning (Optuna)
-  Model registry with production alias
-  CI/CD (GitHub Actions)
-  Containerized serving (FastAPI + Docker)
-  Monitoring and drift detection (Evidently)
-  Automated retraining loop

## Tech stack

Pandas, scikit-learn, implicit (ALS), MLflow, DVC, Feast, Great Expectations, Optuna, FastAPI, Streamlit, Docker, GitHub Actions, Evidently.

## Results so far

Best model: ALS collaborative filtering, precision@10 = 0.3497, tuned via Optuna across 15 trials, tracked and registered in MLflow.

## Setup

```bash
git clone https://github.com/quratalvi11-dotcom/movie-recommender-mlops.git
cd movie-recommender-mlops
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run with Docker (no local setup required)

```bash
docker pull qurat38027/movie-recommender-api:latest
docker run -p 8000:8000 qurat38027/movie-recommender-api:latest
```

Then visit `http://localhost:8000/docs` to try the API.
